## Installation

Node version 16.15.1 or higher is required. Check if you have node installed with- 

```bash
node --version
```

Install the dependencies with -

```bash
npm install
```

Run the application-

```bash
npm run dev
```

Open http://localhost:3000 to view it in your browser.
